### Decisions
- CLAUDE.md 개선 시 `<!-- GSD:*-start/end -->` 마커 내부 내용만 교체하는 비침투적 접근법 채택 (commit `9a0e8a1`, `cdbce0d`)
- README Roadmap에서 Hookify 의존성 완전 제거 — Phase 13 이후 sg-retro로의 전환 문서화 완료

### Surprises
- `skills/sg-retro/SKILL.md`로 정의된 플러그인 스킬이 Skill 도구 목록에 나타나지 않음 — 플러그인 skills 로딩이 commands 로딩과 다르게 동작
- `/init` 명령이 이미 업데이트된 CLAUDE.md에 대해 동일 분석을 두 번 수행함

### Patterns
- 모든 문서 변경을 sg-quick 파이프라인으로 처리하는 패턴 정착
- `/init` 실행 후 발견된 누락 항목을 즉시 두 번째 `/init`으로 보완하는 이터레이션 패턴

### Mistakes
- transcript slug 경로 버그: `tr '/' '-' | sed 's/^-//'` — macOS 경로는 `/`로 시작하므로 변환 후 앞 `-` 존재, `sed 's/^-//'`로 제거하면 잘못된 경로 생성 (sg-retro SKILL.md Step 3b에 동일 버그 존재)
- `super-gsd:sg-retro` Skill 등록 여부를 확인하지 않고 바로 호출 시도
