## v2.10 Requirements

### Grilling (GRILL)

- [x] **GRILL-01**: sg-plan이 discuss 호출 전 메인 컨텍스트에서 한 번에 하나씩 질문하는 grill 단계를 수행한다 (질문 수 무제한)
- [x] **GRILL-02**: 각 질문마다 Claude가 권장 답변을 함께 제시한다
- [x] **GRILL-03**: 코드베이스 탐색으로 답할 수 있는 질문은 사용자에게 묻지 않고 직접 탐색해 해소한다
- [x] **GRILL-04**: 설계 트리 분기를 따라 의존성을 순차적으로 해소한다 (이전 답이 다음 질문을 좌우)
- [x] **GRILL-05**: Claude가 모호함이 해소됐다고 판단하면 합의 요약을 제시하고, 사용자가 '확정' 또는 '추가 질문'으로 종료를 최종 결정한다
- [x] **GRILL-06**: grill 합의 결과(결정·제약)가 gsd-discuss-phase / CONTEXT 입력으로 전달되어 후속 plan에 반영된다

## Out of Scope (honored)

- **gsd-discuss-phase 자체 수정** — Non-invasive 제약 준수. grilling은 sg-plan 메인 컨텍스트의 선행 단계로만 구현, GSD 스킬 미수정 (Agent 프롬프트 inline 주입으로 전달).
- **독립 sg-grill 명령 신설** — 미구현. sg-plan 내부 단계로 한정 (재사용 분리는 차후 검토).
- **Claude 자율 종료(사용자 확인 없음)** — 미허용. 종료는 항상 사용자 확인 게이트(GRILL-05).
