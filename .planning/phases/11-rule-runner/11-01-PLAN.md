# Phase 11 Plan: 자체 Rule Runner

## Objective

hookify 미설치 환경에서 `.claude/hookify.*.local.md` 및 `.claude/sg-rule.*.local.md`를
Python PreToolUse hook으로 직접 실행하는 독립 rule runner를 구현한다.

## Tasks

### Task 1 (type=auto): hooks/rule_runner.py 구현

**파일:** `hooks/rule_runner.py`

hookify PreToolUse hook 로직을 독립 재구현한다. hookify 코드 import 없음.

**핵심 동작:**
1. hookify 설치 감지 (`~/.claude/plugins/cache/claude-plugins-official/hookify/` 존재 시 exit 0)
2. stdin에서 PreToolUse JSON 읽기 (`tool_name`, `tool_input`)
3. `.claude/hookify.*.local.md` + `.claude/sg-rule.*.local.md` 스캔 (중복 name 시 sg-rule 우선)
4. frontmatter 파싱 (name/enabled/event/action/pattern/conditions 두 포맷 모두)
5. event 필터링: `bash` → Bash tool, `file` → Edit/Write/MultiEdit
6. 조건 평가: regex_match, contains, equals, not_contains, starts_with, ends_with
7. 결과:
   - `block` match: `{"hookSpecificOutput": {"hookEventName": "PreToolUse", "permissionDecision": "deny"}, "systemMessage": "..."}`
   - `warn` match: `{"systemMessage": "..."}`
   - no match: `{}`

**검증:**
```bash
python3 hooks/rule_runner.py --help 2>/dev/null; echo "syntax OK: $?"
python3 -c "import ast; ast.parse(open('hooks/rule_runner.py').read()); print('PASS: syntax valid')"
echo '{"tool_name":"Bash","tool_input":{"command":"ls"}}' | python3 hooks/rule_runner.py; echo "exit: $?"
```

### Task 2 (type=auto): hooks/hooks.json에 PreToolUse 추가

**파일:** `hooks/hooks.json`

기존 Stop/SubagentStop 항목 유지하면서 PreToolUse 항목 추가:

```json
"PreToolUse": [
  {
    "hooks": [
      {
        "type": "command",
        "command": "python3 \"${CLAUDE_PLUGIN_ROOT}/hooks/rule_runner.py\"",
        "timeout": 5
      }
    ]
  }
]
```

**검증:**
```bash
python3 -c "import json; d=json.load(open('hooks/hooks.json')); print('PASS: PreToolUse present') if 'PreToolUse' in d['hooks'] else print('FAIL')"
python3 -c "import json; d=json.load(open('hooks/hooks.json')); print('PASS: Stop present') if 'Stop' in d['hooks'] else print('FAIL')"
```
