**bin/setup.js 생성 (per D-03, D-04, D-06, D-07, D-10~D-13):**

`bin/` 디렉토리를 먼저 생성하고, `bin/setup.js` 파일 작성. 구조:

1. **shebang**: 첫 줄 `#!/usr/bin/env node` (D-03)
2. **require**: `fs`, `path`, `util` 표준 모듈만 (외부 의존성 없음, PATTERNS.md 원칙)
3. **ANSI 상수**:
   - `const YELLOW = '\x1b[33m'`
   - `const GREEN = '\x1b[32m'`
   - `const RESET = '\x1b[0m'`
4. **PKG_ROOT 결정**: `const PKG_ROOT = path.dirname(path.dirname(path.resolve(__filename)))` — `bin/setup.js`의 `__dirname`은 `<pkg>/bin/`이므로 `..`이 패키지 루트 (D-07)
5. **parseArgs** (`util.parseArgs`) 사용:
   - `force`: `{ type: 'boolean', default: false }` — --force 플래그 (D-11)
   - `gemini`: `{ type: 'boolean', default: false }` — --gemini 플래그 (D-02/INSTALL-02)
   - `allowPositionals: true` — `install` 서브커맨드 위치 인자 수용
6. **서브커맨드 검증**: positionals[0]이 `'install'`이 아니면 usage 출력 후 `process.exit(1)`
7. **복사 파일 목록 정의**:
   - Codex 기본 항목 (항상 복사):
     - 파일: `.codex/hooks.json` (단일 파일 copyFileSync)
     - 디렉토리: `hooks/` → `hooks/` (fs.cpSync recursive, D-06/D-07)
     - 디렉토리: `.agents/` → `.agents/` (fs.cpSync recursive)
   - Gemini 추가 항목 (`--gemini` 시 추가):
     - 파일: `.gemini/settings.json` (단일 파일 copyFileSync)
8. **복사 로직** (파일/디렉토리별):
   - 소스 경로: `path.join(PKG_ROOT, relativeSrcPath)`
   - 대상 경로: `path.join(process.cwd(), relativeDestPath)`
   - 대상 파일/디렉토리 존재 확인: `fs.existsSync(dest)`
     - 존재 + `--force` 없음: `process.stdout.write(YELLOW + '⚠ ' + relPath + ' already exists — skipping (use --force to overwrite)' + RESET + '\n')` 후 skipped++ (D-10)
     - 존재 + `--force` 있음: 복사 실행 후 `process.stdout.write(GREEN + '✓ ' + relPath + ' (overwritten)' + RESET + '\n')` (D-11)
     - 존재하지 않음: 부모 디렉토리 `fs.mkdirSync(path.dirname(dest), { recursive: true })`, 복사 실행 후 `process.stdout.write(GREEN + '✓ ' + relPath + RESET + '\n')`
   - 디렉토리 복사: `fs.cpSync(src, dest, { recursive: true })` — `--force` 없을 때 대상 경로 존재 시 스킵, 있을 때 overwrite
   - 단일 파일 복사: `fs.copyFileSync(src, dest)`
   - 각 복사 작업을 try/catch로 감싸고 오류 시 stderr에 출력하며 `errors++`
9. **summary 출력** (D-13):
   ```
   \nInstallation complete.\n
     Copied:  {copied} files\n
     Skipped: {skipped} files (already exist)\n
   ```
   errors > 0이면 추가로 `  Errors:  {errors} (see above)` 출력
10. **require.main === module 가드**: `if (require.main === module) { try { main(); } catch(e) { process.stderr.write('[error] ' + e.message + '\n'); process.exit(1); } }`

디렉토리 복사 시 카운팅: `hooks/` 복사를 1개로 카운트하지 말고, 복사된 파일 수를 기준으로 집계하거나, 디렉토리 단위로 카운트하되 일관성 있게 처리. 구현은 discretion에 맡김 (디렉토리 = 1 카운트가 단순).
