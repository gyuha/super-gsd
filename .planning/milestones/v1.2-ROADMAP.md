# Milestone v1.2: Self-Contained Retrospection

**Status:** ✅ SHIPPED 2026-05-21

## Overview

hookify에 의존하던 회고(retrospection) 단계를 super-gsd 내장 `sg-retro` Skill로 완전히 대체한다. 6가지 lens를 지원하는 내장 회고 Skill, hookify 미설치 환경에서도 동작하는 자체 rule runner, weighted top-N lessons 랭킹 시스템을 구축하고, hookify 의존성을 모든 문서와 명령어에서 제거한다.

## Milestone Summary

**Key Decisions:**

- sg-retro Skill을 hookify 외부 의존 없이 self-contained로 구현 (D-11: transcript 스캔은 Phase 10 ANALYZER scope로 분리)
- 6개 lens: SSC, 4Ls, DSPM, Sailboat, Five Whys, Conversation Analyzer — multiSelect로 한 번에 여러 lens 선택
- rule_runner.py hookify 설치 감지 → exit 0 skip (hookify 기존 사용자 호환성 유지)
- weighted ranking: score = 0.4×freq + 0.4×recency + 0.2×severity
- HANDOFF.md stage 값 "hookify" 보존 (sg-status 라우팅 호환성, D-02 Phase 13)

**Issues Resolved:**

- macOS BSD grep PCRE 비캡처 그룹 미지원 → ERE + sed 조합으로 대체
- sg-complete Step 번호 물리적 순서 불일치 (Step 0.5 → Step 1.3으로 재명명)
- lessons_ranker.py warn 메시지 stdout 출력 → stderr로 수정

**Issues Deferred:**

- MIGRATION-02: hookify 미설치 환경 e2e manual checklist (다음 수동 실행 시 검증)
- LESSONS-01: sg-retro append 동작 검증 (기구현이나 통합 테스트 미진행)

**Technical Debt:**

- sg-status: "hookify" stage 표시가 실제로는 sg-retro임을 반영하지 않음 (하위 호환성 우선)
